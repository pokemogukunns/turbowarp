ターボワープのプライバシーポリシー

2023年8月21日更新

TurboWarpプロジェクトは、あなたのプライバシーを尊重します。 すべてのウェブサイトはこれを言いますが、私たちはそれを意味しています：私たちはあなたのデータに興味がありません。完全な透明性のために、この文書には、あなたのデータが収集または共有される可能性のあるすべての場所がリストされています。

プロジェクトの読み込み

別のウェブサイトからプロジェクトをロードする場合、そのウェブサイトのプライバシー慣行の対象となります。たとえば、スクラッチからプロジェクトをロードする場合、スクラッチプライバシーポリシー。 Scratch からプロジェクトを具体的にロードする場合、Scratch の API の一部として直接アクセスが許可されていないため、プロジェクト ID も TurboWarp と共有されます。キャッシュ用のプロジェクトIDを簡単に記録する場合があります。

Running projects

クラウド可変サーバーに接続する場合、プロジェクトIDとユーザー名は14日間記録される場合があります。クラウド変数のデータは、同時に接続されている他の人に送信されます。カスタムクラウド可変サーバーは、当社の制御外です。

一部の拡張機能は、機能するために外部 API と通信する必要があります。たとえば、翻訳とテキスト読み上げの拡張機能は、ScratchプライバシーポリシーでカバーされているScratch APIに依存しています。パフォーマンスを向上させるために、一部の拡張機能は代わりにTurboWarpのAPIにアクセスし、要求（たとえば、翻訳されるテキスト）とScratchに転送した後の結果（たとえば、翻訳されたテキスト）の両方がキャッシュされる可能性があります。

公式拡張機能ギャラリーのほとんどの拡張機能は、プロジェクトが拡張機能を使用して信頼できないWebサイトにアクセスしようとすると、許可を求めますが、これが100%信頼できることを保証することはできません。

コンピューター上のURLやファイルなど、公式ギャラリー以外の場所からカスタム拡張機能をロードする場合、エディターは拡張機能をロードする許可を求めます。このダイアログを承認すると、拡張機能は公式拡張機能に表示される許可ダイアログをバイパスできます。

設定

多くのTurboWarpウェブサイトは、addon設定やpackager設定などの設定をブラウザに保存します。

集計データ

1日に特定のサイトを利用する人の数など、匿名の集計データを記録する場合があります。

turbowarp.org

ランダムに生成されたユーザー名がブラウザに保存される場合があります。これをカスタムユーザー名に置き換えることができます。ランダムに生成されたユーザー名は、クラウド変数サーバーに送信される前に匿名化されます（ランダムな部分は削除されます）。

ターボワープパッケージャー

TurboWarp Packagerは、利便性とパフォーマンスのために、設定を保存し、Electronなどの大規模なダウンロードをブラウザにキャッシュする場合があります。

カスタム拡張機能を使用してプロジェクトをパッケージ化する場合、パッケージング業者はそれらのWebサイトにアクセスして拡張機能をダウンロードする場合があります。

TurboWarp Packagerによって生成されたプロジェクト

パッケージ化されたプロジェクトには、プロジェクトを実行するためのすべてのスクリプト、衣装、およびサウンドが含まれています。パッケージャーは、すべてのカスタム拡張機能も含めようとしますが、場合によっては、パッケージ化されたプロジェクトが実行されるたびに特定の拡張機能をダウンロードする必要がある場合があります。

サードパーティのカスタム拡張機能、カスタムクラウドホスト、カスタムJS、カスタムCSSなどの高度な機能は、当社の制御外です。

TurboWarpエディターとは異なり、パッケージ化されたプロジェクトのカスタム拡張機能は、Webサイトに連絡する前に許可を求めないことに注意してください。

配布するファイルのプライバシー慣行について、エンドユーザーに通知する責任があります。今後変更される可能性があるため、プロジェクトのプライバシーポリシーとしてこのドキュメントにリンクしないでください。自分で書かなければなりません。
